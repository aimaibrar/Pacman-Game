# 👻 Pacman Console Game (Java OOP Project)

A simple console-based Pacman game built using Java and Object-Oriented Programming. The player controls Pacman inside a maze while avoiding a randomly moving ghost. The goal is to survive as long as possible.

## 🎮 Features
- OOP-based design (Pacman, Ghost, GameBoard, Food)
- Player-controlled movement using < > ^ v
- Random ghost movement
- Maze-style board
- Collision detection (Game Over when ghost catches Pacman)
- Console-based interface

## 🕹️ Controls
- < Move Left  
- > Move Right  
- ^ Move Up  
- v Move Down  

## 🎯 Objective
Survive in the maze by avoiding the ghost 👻. (Future update: collect dots for score)

```bash

java Main
